# TermXT 
TermXT is a terminal direct messaging app made with python and express.js.

## Setup
- install program with pipx
    ```sh
    pipx install git+https://github.com/Sivayogeith/termxt.git
    ```
- register
    ```sh
    termxt register  
    ```
- chat
    ```sh
    termxt chat  
    ```

## Commands
- login
    ```sh
    termxt login  
    ```
- register
    ```sh
    termxt register  
    ```
- chat
    ```sh
    termxt chat  
    ```
- logout
    ```sh
    termxt logout  
    ```
